# Calculadora de Partidas Rankeadas

## Objetivo

Crie uma função que recebe como parâmetro a quantidade de vitórias e derrotas de um jogador. O objetivo é calcular o saldo de partidas ranqueadas através do cálculo:


Com base no número de vitórias, o jogador será classificado em um nível.

## Regras de Classificação

- Vitórias < 10 → **Ferro**
- Vitórias entre 11 e 20 → **Bronze**
- Vitórias entre 21 e 50 → **Prata**
- Vitórias entre 51 e 80 → **Ouro**
- Vitórias entre 81 e 90 → **Diamante**
- Vitórias entre 91 e 100 → **Lendário**
- Vitórias ≥ 101 → **Imortal**

## Tecnologias/Conceitos a serem utilizados

- Variáveis
- Operadores
- Laços de repetição
- Estruturas de decisão
- Funções

## Saída Esperada

Ao final, deve ser exibida uma mensagem no formato:

O Herói tem de saldo de {saldoVitorias} está no nível de {nivel}
